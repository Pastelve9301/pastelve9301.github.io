  • Manage buy-binds and utility throw presets.
  • Back up your existing configuration files safely.

  1. Visit the official download page.
  2. Locate the latest release section on the right side of the page.
  3. Download the file ending in .exe to your desktop.

💻 System requirements

This application works on all modern versions of Windows. It requires minimal system resources. You need approximately 50MB of hard drive space and an active installation of Counter-Strike 2 on your primary drive.

🛠️ How to use profiles

When you open the app, it attempts to find your game installation automatically. If it fails, use the settings menu to point the app to your Steam folder. Once the directory is set, you can create your first profile.

Click the "New Profile" button to name your setup. Add your preferred commands into the text box provided. When you click "Apply," the tool updates your local config file. You can see your changes take effect during your next match.

🛡️ Safety

The tool only modifies local text files within the Counter-Strike 2 folder. It does not interact with protected game memory or network packets. This approach keeps your account safe from automated game bans. Always keep a backup of your original config file before applying large batches of changes.

❓ Troubleshooting

If the app does not detect your game, ensure Steam is running. If settings do not apply in-game, ensure you have set your configuration to read-only mode off before using the tool. Most problems result from incorrect file paths in the settings menu.
